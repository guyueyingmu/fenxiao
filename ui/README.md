# elementui

>  Vue2.0 +  element-ui

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
//后台开发
npm run dev

//前台开发
npm run devapp 


# build for production with minification
//后台打包

//打包库
npm run admindll

//打包源码
npm run admin


//前台开发

//打包源码
npm run app

//打包库
npm run appdll
